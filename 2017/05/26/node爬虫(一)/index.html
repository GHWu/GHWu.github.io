nodejs爬虫(一)

  • nodejs
  • 爬虫

开发环境

–nodejs
具体安装教程：教程

建立工程

在新建一个文件夹目录下使用npm init初始化工程，此时需要填写一些信息也可以一路回车

创建完会在文件夹下生成一个package.json文件。然后需要安装三个第三方包：npm install cherrio -save.npm install superagent-charset -save.npm install superagent -save。(另外还会用到fs，但fs是内置包不用额外安装)

P.S：-save是将项目对该包的依赖写入到package.json文件中。

安装完包后在项目文件里新建一个data文件夹用来储存爬取的数据。并在项目文件目录下创建一个index.js文件。到此工程建立完毕。

Code

打开index.js文件。

文件开头先引入包：

var http = require('http');
var fs = require('fs');
var cheerio = require('cheerio');
var charset = require('superagent-charset');
var superagent = charset(require('superagent'));

定义网址和时间(时间后面用来命名文件)：

var url = "http://www.dy2018.com"; 
var date = new Date();

定义函数：

function startRequest(x){
    //使用superagent防止得到乱码
    superagent.get(x).charset().end(function (err, res) {
      var context = "";
    //cheerio模块解析html
    var $ = cheerio.load(res.text);
    //获取所有的电影链接及名称
    var video = $('.co_content222 a').toArray();

    //后面用时间命名txt文件    
    var year = date.getFullYear();
    var month = date.getMonth()+1;
    var day = date.getDate();    
    var time = year + "-" + month + "-" + day;    

    //把video数组中的链接循环读出
    for (var i = 0; i < video.length; i++) {
        var link = 'www.dy2018.com'+video[i].attribs.href;
        var title = video[i].attribs.title;
        context = context + title + "------------------" + link + "**************************"; 
    }
    //将电影名称和链接写入到data文件夹中的txt文件中并以当前时间命名
    fs.appendFile('./data/' + time + '.txt',context,'utf-8',function(err){
        if(err){
            console.log(err);
        }
    });
});
}

最后执行函数：

startRequest(url);

运行

cmd进入控制台cd进入项目目录后node index.js就可以让程序跑起来了。
